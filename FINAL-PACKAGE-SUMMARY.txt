GOOGLE ANALYTICS CONVERSION GOALS:
===================================

CREATE THESE GOALS:
1. Booking Completed
   - Type: Destination
   - Destination: /thank-you.html
   - Value: Varies by booking

2. Contact Form Submitted
   - Type: Event
   - Category: form
   - Action: submit

3. Phone Number Clicked
   - Type: Event
   - Category: contact
   - Action: phone_click

4. Language Changed
   - Type: Event
   - Category: language
   - Action: change

HOW IT WORKS:
=============

LANGUAGE SWITCHING:
1. User on English page
2. Clicks "FR" button
3. Browser goes to yoursite.com/fr/
4. Loads complete French HTML file
5. Everything is French (not translated, but a different file!)

BOOKING FLOW:
1. User selects dates on homepage
2. Dates saved to localStorage
3. User clicks car "Book Now"
4. Booking page loads with dates pre-filled
5. User completes form
6. Redirects to /thank-you.html
7. Google Analytics tracks conversion

ADMIN PANEL:

2. Dashboard shows stats
3. Bookings tab: Mark Paid/Denied

5. Edit Cars tab: Update images & prices
6. Send Form tab: Create bookings
